# 数学知识星图项目交接

## 项目目标

建立一个面向中国数学教学的可交互知识图谱。当前已按最新要求收缩为：

- 初中数学：较完整的核心知识网络
- 高中数学：只保留高一中最关键、最适合与初中衔接的函数、三角、向量与几何主线
- 优先保证整体稳定运行，再逐步补细节

## 用户原始要求与最新范围

- 要做成可发散、可连接的数学知识图谱，不只是一张函数样板图。
- 每个大知识点要能连出多个小知识点，并与跨章节知识建立关系。
- 用户已授权后续直接执行，不需要反复确认。
- 最新范围缩减要求：
  - 后续内容要控制总量，减少工作量
  - 高中内容只保留一部分
  - 只做高一知识点

## 关键决定

- 不再按“全高中数学”扩张，改为“初中全量主干 + 高一高价值主线”。
- 保留统一 schema 和关系规则，不为缩范围牺牲数据质量。
- 前端优先做“稳定接库”，避免数据增长后页面崩溃。
